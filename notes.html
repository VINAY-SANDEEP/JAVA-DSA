• Class : A blueprint or template in Java that defines the structure and behavior (fields and methods) of objects.
  • Object : An instance of a class that holds data and can perform actions defined by its class.
  • Constructor : A constructor in Java is a special method used to initialize objects, typically called when an object of a class is created, and it has the same name as the class.
  • This(keyword) : The this keyword in Java is used to refer to the current instance of a class, helping to distinguish between instance variables and parameters, or to call other constructors within the same class
  • Wapper clases : A wrapper class in Java provides an object representation of primitive data types, allowing primitives like int, char, double, etc., to be treated as objects (e.g., Integer, Character, Double).
      • Byte (for byte) , Short (for short), Integer (for int), Long (for long), Float (for float), Double (for double), Character (for char), Boolean (for boolean),
  • Packages : In Java, a package is a namespace that groups related classes and interfaces together, helping to organize code, avoid name conflicts, and control access levels.
  • it is also know as folder
  • static : used to indicate that a method, variable, or block belongs to the class rather than instances of the class, allowing it to be accessed without creating an object of the class.
  • Non-static : a non-static method or variable belongs to an instance of a class, meaning it requires creating an object of the class to be accessed or used.
      • Key Differences:
    • Static Method: Called using the class name (no object required).
    • Non-Static Method: Called using an object of the class (object required).
  • Innerclass :Java is a class defined within another class, used to logically group classes that are only used in one place, or to access the outer class's members (even private ones).
    1. Nested Inner Class: A regular class defined inside another class.
    2. Method-local Inner Class:Defined within a method and can only be accessed within that method.
    3. Anonymous Inner Class:A class without a name, used to instantiate objects with modifications to existing classes or interfaces.
    4. Static Nested Class:A static class inside another class, which can be accessed without an instance of the outer class.
  • Inheritance : Java is a mechanism where one class (subclass/child) acquires the properties and behaviors (fields and methods) of another class (superclass/parent), promoting code reuse and establishing a hierarchical relationship.
  • Encapsulation : Java refers to the bundling of data (variables) and methods that operate on the data into a single unit (class)
  • Abstraction : Java is the concept of hiding the complex implementation details and showing only the essential features of an object, primarily achieved using:
    1. Abstract Classes: Classes that cannot be instantiated and may contain abstract methods (methods without implementation).
    2. Interfaces: Purely abstract types that define a contract which classes must follow, without specifying how the methods should be implemented.
  • Polymorphism : Java is the ability of a single action or method to behave differently based on the object or context.
    1. Compile-time (Method Overloading): Multiple methods in the same class with the same name but different parameter lists.
    2. Runtime (Method Overriding): A subclass provides a specific implementation of a method that is already defined in its superclass, allowing dynamic method dispatch.
